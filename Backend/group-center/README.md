# Group Center Backend

### 语言及构建工具

- Java
- Kotlin
- Gradle(Kotlin DSL)

### IDE及工具

- Alibaba DragonWell JDK 21
- Jetbrains Intellij IDEA
- Jetbrains DataGrip
- PDManer

### 框架

- Spring Boot

### 数据库

- Alibaba Druid
- MyBatis
- MyBatis Plus

- MySQL
- Alibaba OceanBase

- Tencent Kona Crypto(国密SM3/SM4算法)

### 消息推送

- 企业微信(WeCom)
- 飞书(Lark)
