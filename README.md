# Group Center

课题组消息中心

**Author:**

* [Haomin Kong](https://github.com/a645162)

本项目仅为作者的Java前后端分离**学习**使用！

**目标:**

**尽可能**使用国产工具(框架)。

## 主要功能

- GPU使用记录存储
- GPU使用情况统计
- GPU任务结束通知推送
